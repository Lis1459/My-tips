# My tips

## Описание

SPA-приложение для управления контрактами с возможностью просмотра статистики и аналитики в виде графиков.

## Краткая функциональность

- Авторизация пользователя
- Просмотр существуюзих контрактов
- Добавление новых контрактов
- Изменение/удаление существующих контрактов
- Отображение статуса контракта (активный, ошибочный, приостановленный, истекший)
- Страница деталей с аналитикой и графиками
- Адаптивный интерфейс

## Список зависимостей

Основнвые зависимости:

- vue
- vue-router
- vuex
- echarts
- vite

## Структура проекта

В папке source(src) есть несколько папок, которые каждая отвечает за свою задачу:

- assets — папка с картинками, иконками и базовыми стилями
- components — тут храняться кастомные компоненты
- router — папка с настройкой vue-router
- store — папка, где настроено vuex-хранилище
- views — здесь лежат страницы приложения.

## Инструкция по запуску

1. Клонируйте репозиторий

`git clone https://github.com/Lis1459/My-tips.git`

2. Перейдите в папку с проектом

`cd My-tips`

3. Запуск в режиме разработки

`npm run dev`

После запуска проект будет доступен на по адресу:

`http://localhost:5173`

4. Конечная сборка проекта (для продакшена)

`npm run build`
